# rtc-demo
WebRTC demo with vanila JS

## How to use

1. Clone project

```bash
git clone https://github.com/escalopa/rtc-demo.git
cd rtc-demo.
```

2. Add firebas configurations & fill env variables values by the firebase app credentials
```bash 
cp .env.example .env
```

3. Run the app
```bash
npm i
npm run dev
```
